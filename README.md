# HuskyZoom - Video Chat App

### Summary

WebRTC와 Socket IO 기능을 버무려 만든 비디오 채팅 웹 앱입니다.

2021년 최신 프론트엔드 환경을 많이 체득하려고 노력했습니다.

기존의 React + Webpack + Babel + Terser 조합을 버리고 이번엔 이런 조합을 선택해 보았습니다.
